# Restaurant Site

Site vitrine statique pour un restaurant gastronomique fictif.  
Ce projet illustre :

- une structure HTML propre
- Tailwind CSS bien configuré
- JavaScript modulaire pour interactions
- animations et effets 3D sur les cartes

## Technologies utilisées

- HTML
- Tailwind CSS
- Vanilla JavaScript

## Utilisation

1. `npm install`  
2. `npm run build`  
3. Ouvrir `index.html` dans un navigateur.
